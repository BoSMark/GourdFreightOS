## Notes on Agent Types (to be defined in Workshop)

GOURD Freight at this stage is a very small company. When building agent specs in Workshop, the likely useful agent types are:

- **Route and Rate Research Agent** (Operational): researches carrier options, surfacing lowest-cost and lowest-carbon routes for account manager review
- **Carbon Reporting Agent** (Operational): generates per-shipment carbon reports for client reporting
- **Client Relationship Agent** (Operational): drafts client communications, tracks booking status, flags delivery exceptions for account manager review
- **Commercial Pipeline Agent** (Operational): tracks new client pipeline, conversion rates, and account manager book growth

All of these support human account managers; none of them should make decisions autonomously without human review at this stage.
